# `.claude/commands` (ois-cfa)

Каталог для проектно-специфичных команд агентов внутри `ois-cfa`.

Примеры будущих команд:
- `nx-plan` — разложить конкретный `tasks/NX-0X-*.md` на подшаги и DoD.
- `nx-daily-report` — собрать daily-report с привязкой к NX-таскам и веткам.
- `nx-test-matrix` — построить матрицу тестов по сервису/таску.

Сейчас создаётся только каркас; конкретные команды можно добавлять по мере стабилизации NX-00/NX-0X процессов.
