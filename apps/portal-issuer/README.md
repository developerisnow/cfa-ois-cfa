# Portal Issuer

Next.js 15 портал для эмитентов.

## Pages

- `/dashboard` - Дашборд
- `/issuances` - Список выпусков
- `/issuances/create` - Создать выпуск
- `/issuances/[id]/publish` - Опубликовать
- `/issuances/[id]/close` - Закрыть
- `/reports` - Отчёты
